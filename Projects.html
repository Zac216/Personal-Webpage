Location Reminders (Android App)
Location Reminders is an application that I developed for Android as a
summer project. This project was not for a class, was not suggested by
anyone, and was only worked on by me in a little over one summer for
a couple hours per day.
This application allows the user to set a reminder for themselves to be
activated when they reach a location. Ex, "The next time that I go to
Subway, remind me that I have a gift card for there."
It utilizes a SQLite database to save the information and checks less
frequently if the user is farther away to save battery and processing
power. It can be found at GitHub.com/zac216.

Nearest Neighbor Algorithm Calculator
For my Discrete Mathmatics class (MAT 226), I created a program to
find the nearest neighbor path and length given an edge set. The
code is about 300 lines long and was written in C#. An example of
when this program would be useful would be if you wanted to take a
road trip and visit several cities along the way while taking the
shortest and most efficient route.
